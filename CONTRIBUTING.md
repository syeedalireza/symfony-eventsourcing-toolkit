# Contributing to Event Sourcing Toolkit

## Development Setup

```bash
composer install
docker-compose up -d
make test
```

## Code Quality

- Follow PSR-12
- Run `make quality` before committing
- Write tests for new features
- Update documentation

## Pull Requests

1. Fork the repo
2. Create feature branch
3. Write tests
4. Ensure all checks pass
5. Submit PR

Thank you for contributing!
